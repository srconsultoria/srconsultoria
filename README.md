# S&R Consultoria — Site (Cloudflare Pages / GitHub)

Site estático (HTML/CSS/JS). Pronto para publicar em **Cloudflare Pages** conectado ao GitHub.

## Estrutura
- `index.html`
- `css/style.css`
- `js/main.js`
- `img/` (imagens otimizadas em WebP)

## Publicar no Cloudflare Pages
- Framework: **None**
- Build command: *(vazio)*
- Output directory: `/`

## Desenvolvimento
Abra `index.html` no navegador (ou use uma extensão de servidor local no VS Code).

## Contato
Atualize o número do WhatsApp em `js/main.js` (função `openWhatsApp`) se necessário.
